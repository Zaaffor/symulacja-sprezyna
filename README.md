# symulacja-sprezyna

Program oblicza numerycznie tłumienie drgania obiektu zawieszonego na sprężynie za pomocą metody Eulera. Na ten moment parametry są podawane w kodzie programu a sama wizualizacja jest bardzo prosta. Niebieska linia reperezentuje sprężynę, biały prostokąt ciało a czerwone linie wyznaczają osie symetrii okna animacji.
